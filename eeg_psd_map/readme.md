### plot_eeg.py
在plot_eeg.py中实现了对于eeg多文件的读取，读取的文件格式为.fif文件（为去除眼电后的脑电信号）  
save_psd函数用于保存脑电信号epochs的功率谱密度脑地形图  
save_channel_psd函数用于保存脑电信号epochs的某个导联的功率谱密度（目前只实现了查看脑电信号功率谱密度），在该函数中需要确定选择的导联位置，根据需要修改channelnames数组。  

提取脑电信号特征过程:  
1.截取同一类型的脑电信号，需要确定所需要的脑电信号的标签和截取的长度。  
2.将同一类型的脑电信号进行叠加，得到这一类型的脑电信号的epochs合集  
3.根据epochs合集进行功率谱密度分析，确定影响这一类型的具体脑区  
4.选择合适的脑区导联，绘制各个导联的功率谱密度函数，分析其特征
